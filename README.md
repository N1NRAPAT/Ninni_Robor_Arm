Ninni Robot Arm : summer project - started : 21/06/2026

This is set to be an open source for anyone looking for simple 5 joints inverse kinemetic integrated to robotic arm which use STM32 as main 
controller of electronice devices and the rest of image processing and prediction are created real time as backgound in pc. Therefore this 
project are base on ROS2 , Docker container and CUDA tools to concentrated outcome.
